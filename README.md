## 🚀 What I am building right now

### 🛍️ SmartDeal Stylist (active)
A personal AI stylist and deal alert agent built out of pure frustration with how 
hard it is to track deals on women's apparel from trusted brands like Zara, H&M, 
and Mango. Think Slickdeals but actually built for fashion.

- Scrapes and aggregates deals from targeted, trusted shopping sites
- Recommends outfits based on your style, body type, and past searches using RAG
- Sends email alerts when monitored items drop in price
- Future: autonomous purchasing agent with smart return policy checking

👉 [View Project](https://github.com/suprithakonaje/smartdeal-stylist)

### 📰 Financial News Chatbot
A RAG based chatbot that retrieves and summarises financial news using FAISS vector 
search, HuggingFace LLMs, and a Flask web interface. Includes unit tests and 
evaluation using ROUGE and BERTScore metrics.

👉 [View Project](https://github.com/suprithakonaje/financial-news-chatbot)
